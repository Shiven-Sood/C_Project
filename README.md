# Personal Diary Management System

## Project Description
The Personal Diary Management System is a console-based application developed using the C programming language.  
It allows users to securely store, view, and edit diary entries with password protection and basic encryption.

The project demonstrates the use of file handling, string manipulation, functions, and menu-driven programming in C.

---

## Features
- Password-protected access
- Add diary entries with date
- View and edit entries by date
- Caesar Cipher encryption for security
- File-based permanent storage

---

## Technologies Used
- Programming Language: C
- Compiler: GCC / Turbo C
- Operating System: Windows / Linux
